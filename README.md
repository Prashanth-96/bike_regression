OBJECTIVE:
  To predict the number of bikes that are to be rented based on specific conditions

DATASET:
   The dataset has been taken from https://archive.ics.uci.edu/ml/datasets/Seoul+Bike+Sharing+Demand
The dataset contains count of public bikes rented at each hour in Seoul Bike haring System with the corresponding Weather data and Holidays information.

THIS PROJECT HAS
 1.Bike_rent.ipynb : A python jupyter notebook file which as a detailed analysis,visualisation and model building.
 2.Bike_rent.py    : A python file which has a Flask implementation.
 3.templates :
         home.html : home page for the flask app
 4.static   :
        css :
        styles.css :A stylesheet for home.html
        bike_img2.jpg : the backgroud image.

 5.bike_rent.pkl : A pickle file consisting the model
